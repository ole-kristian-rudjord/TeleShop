# TeleShop

## Setup

Make sure to install the dependencies:

```bash
# npm
npm install

# pnpm
pnpm install

# yarn
yarn install

# bun
bun install
```

## Development Server

Start the development server on `http://localhost:3000`:

```bash
# npm
npm run dev

# pnpm
pnpm run dev

# yarn
yarn dev

# bun
bun run dev
```

## Technologies Used

- TypeScript
- Vue 3 (with Composition API)
- Nuxt 3
- Vuetify 3
- Sass
- Font Awesome
